# VK Poll Payment Bot

Бот для беседы ВК, который создает сбор денег по ответам в опросе.

## Что умеет

- Команда `/getMoney <текст_для_оплаты>` (ответом на сообщение с опросом).
- Команда `/ping` для проверки, что бот онлайн (`pong` в ответ).
- Команда `/help` показывает краткую инструкцию по командам.
- Команда `/createGame <название>` создает опрос в чате:
  - ответы: `Иду`, `+1`, `+2`, `+3`, `Не иду`
  - множественный выбор
  - неанонимный
  - можно менять ответ
  - без ограничения по времени
  - по умолчанию выбирается зеленый фон (если VK возвращает доступные фоны)
- Из опроса учитываются варианты:
  - `Иду` -> сам участник.
  - `+1`, `+2`, `+3`, ... -> сколько приглашенных у автора голоса.
  - Если пользователь выбрал несколько `+N`, суммы складываются.
- Бот публикует сообщение со списком неоплативших и кнопками:
  - `Оплатил` -> убирает самого нажавшего из списка игроков.
  - `Оплатил приглашенный` -> уменьшает счетчик приглашенных у нажавшего.

## Подготовка

1. Создайте сообщество/бота ВК и получите `VK_TOKEN`.
2. Получите пользовательский токен `VK_USER_TOKEN` (именно user token), он нужен для чтения голосов в опросе (`polls.getVoters`).
3. Включите Callback API и Long Poll для группы.
4. Дайте права group-токену минимум на: `messages`.
5. Разрешите боту писать в беседу.

## Установка

```bash
cp .env.example .env
# впишите VK_TOKEN и VK_USER_TOKEN в .env
npm install
npm start
```

## Использование

1. В беседе есть опрос с вариантами `Иду`, `+1`, `+2`, `+3`.
2. Ответьте на сообщение с опросом командой:

```text
/getMoney 500р, перевод на 8-999-123-45-67
```

3. Бот отправит сообщение со списком тех, кто еще не оплатил.
4. Участники нажимают кнопки по факту оплаты.

Создать новый игровой опрос:

```text
/createGame Волейбол в четверг 20:00
```

## Хранилище

Состояние хранится локально в `data/state.json`.
